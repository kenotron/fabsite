  The `merge-styles` library provides utilities for loading styles through javascript. It is designed to make it simple to style components through javascript. It generates css classes, rather than using inline styling, to ensure we can use css features like pseudo selectors (:hover) and parent/child selectors (media queries).

  The library was built for speed and size; the entire package is 2.62k gzipped. It has no dependencies other than `tslib`.

import { mergeStyles, mergeStyleSets } from 'office-ui-fabric-react';

### MergeStyles

MergeStyles returns a className (like `css-0`) which can be used anywhere.

```js
const className = mergeStyles({
  backgroundColor: 'red',
  color: 'white'
});

render(<div className={className}>hi</div>);
```

### MergeStyleSets

Often you want to return a set of styles. This is where you'd use `mergeStyleSets` which will return:

```js
{
  root: 'root-0',
  title: 'title-0',
  button: 'button-0'
}
```

Apply these classNames to your components and the styles will be applied.

```js
let classNames = mergeStyleSets({
  root: { background: 'red' },
  title: { backgroundColor: 'green' },
  button: {
    border: 'none',
    margin: 10,
    selectors: {
      '&:hover': {
        background: 'purple'
      }
    }
  }
});

render(
  <div className={classNames.root}>
    <h1 className={classNames.title}>Hello World</h1>
    <button className={classNames.button}>Click Me</button>
  </div>
);
```
